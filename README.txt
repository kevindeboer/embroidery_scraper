1. Download Python: https://www.python.org/downloads/release/python-374/
Zorg dat 'add to PATH' aangevinkt is, dan 'gewone installatie' niet 'custom'

2. Shift-rechtermuis in deze map, en selecteer 'Powershell-venster hier openen' of 'Commando-venster hier openen' oid

3. Type: "pip install -r requirements.txt" (enter)

4. Open bestand 'config.json' (in bijv notepad) en pas de waarden aan naar eigen wens. 
Pas de 'destination_folder' aan naar bijvoorbeeld 'C:\Users\Mikky\Documenten\...'
Vul voor elke website je inloggegevens in

5. Om het script te starten, type in het commandovenster van stap 2: "python main.py"

Herhaal stap 2 en 5 elke dag
